“the earliest time at which all members are connected” is the time when the number of connected component is 1.

using WeightedQuickUnionUF from algs4.jar library

main implementation:

public static void main(String[] args) {
		WeightedQuickUnionUF uf = new WeightedQuickUnionUF(10);
		StdOut.println(uf.count() == 10);
		uf.union(0, 1);
		StdOut.println(uf.count() == 9);
		uf.union(2, 3);
		StdOut.println(uf.count() == 8);
		uf.union(0, 2);
		StdOut.println(uf.count() == 7);
		uf.union(1, 3);
		StdOut.println(uf.count() == 7);
}
